# isakakotaro
## 프로젝트 개요
- isakakotaro 홈페이지

## 들여쓰기
- HTML: 4tab
- SCSS: 2tab
- JavaScript: 4space

## 프로젝트 설정
```
clone git+https://github.com/pigjh1/isakakotaro.git
cd isakakotaro
npm install gulp -g # gulp이 설치되지 않았을 때
npm install webpack -g # webpackdl 설치되지 않았을 때
npm i(install) : 설치
```

## task 명령어
- gulp : 기본 실행
- gulp icon : fontello icon 생성
- gulp dist : dist경로에 배포용 파일 생성
